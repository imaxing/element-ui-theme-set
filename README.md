# 动态设置elementui主题

### install

```bash
npm i @iamgx/element-ui-theme-set -S
  
```

### usage


```javascript
import modifyElementUiTheme from '@iamgx/element-ui-theme-set'
modifyElementUiTheme({ theme: '#f40f40' });
```
